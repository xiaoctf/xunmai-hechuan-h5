## 二、本项目采用的设计取舍

### 1. 不做学校活动宣传页

实体物料和页面首屏优先呈现“数智赋能非遗，文创助农振兴”的地方文化与产品推广属性。团队与“三下乡”成果属性后置到底部落款，避免二维码扫开后第一眼像活动报名页或学校宣传页。

### 2. 移动端做 H5 故事流

移动端主要面向扫码访问，采用 9 屏纵向叙事：首页、项目导语、非遗资源库、匠人故事、实践链路、实践路线、助农产品、文创展示、团队落款。

### 3. 桌面端做展览导览站

桌面端主要面向老师电脑查看、投屏汇报和后续展示，采用横向信息密度更高的展览站结构：顶部导航、沉浸式首屏、栏目化内容、路线板块、成果落款。

### 4. 自动识别设备

入口页 `index.html` 自动识别访问设备：

- 手机或触摸设备进入 `mobile.html`。
- 桌面设备进入 `desktop.html`。
- 保留锚点，例如 `index.html#route` 会自动进入对应端的路线板块。

### 5. 核心非遗对象先锁定，真实材料后补

根据最新版策划书，当前核心对象锁定为合川桃片、青草坝萝卜干/卷、峡砚、泥塑、木雕。页面先展示项目逻辑和内容架构，实地走访和资料核验后，再补充真实传承人访谈、工艺视频、游客试吃数据和农户走访记录。

## 三、后续可升级方向

1. 用真实照片替换当前抽象场景视觉。
2. 给每个非遗项目建立单独详情卡。
3. 增加传承人音频或短视频入口。
4. 将路线图接入真实地图截图或手绘地图。
5. 将问卷二维码替换为正式问卷链接，并把统计结果做成看板。
